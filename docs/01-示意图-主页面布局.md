# 主页面布局示意图

这是图片压缩工具的主页面布局，展示了应用启动后用户看到的第一个界面。

```
+----------------------------------------------------------------------+
|                     图片压缩工具 - Image Compressor                    |
+----------------------------------------------------------------------+
| [☀️] [🌙] |                                          | [设置 ⚙️] [帮助 ?] |
+----------------------------------------------------------------------+
|                                                                      |
|  +------------------------------------------------------------------+  |
|  |                                                                  |  |
|  |                          拖拽图片到这里                            |  |
|  |                              或者                                |  |
|  |                     [ 点击选择图片文件 ]                          |  |
|  |                                                                  |  |
|  |            支持JPG、PNG、GIF、WEBP等格式，最大30MB                |  |
|  |                                                                  |  |
|  +------------------------------------------------------------------+  |
|                                                                      |
|                       [ 上次处理的图片列表 ]                          |
|                                                                      |
|  +-------------+  +-------------+  +-------------+  +-------------+  |
|  |             |  |             |  |             |  |             |  |
|  |  缩略图 1    |  |  缩略图 2    |  |  缩略图 3    |  |  缩略图 4    |  |
|  |             |  |             |  |             |  |             |  |
|  |  已压缩 50%  |  |  已压缩 35%  |  |  已压缩 60%  |  |  已压缩 45%  |  |
|  +-------------+  +-------------+  +-------------+  +-------------+  |
|                                                                      |
|                     [ 批量上传 ] [ 清除全部 ]                         |
|                                                                      |
+----------------------------------------------------------------------+
|                           状态栏/提示信息                             |
+----------------------------------------------------------------------+
```

## 主要区域说明

1. **顶部导航栏**: 包含应用名称、主题切换按钮、设置和帮助按钮
2. **上传区域**: 支持拖放和点击选择文件，显示支持的格式和文件大小限制
3. **历史记录区域**: 显示最近处理过的图片缩略图和压缩率
4. **底部操作栏**: 包含批量上传和清除操作按钮
5. **状态栏**: 显示当前操作状态和提示信息

## 实现优先级

- **第一阶段**: 上传区域、基本导航功能
- **第二阶段**: 历史记录展示、状态栏
- **第三阶段**: 主题切换、设置和帮助功能
